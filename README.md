FlashForge — Карточки для запоминания (Флеш-карты) на 7 языках
FlashForge — это коллекция из семи независимых реализаций системы флеш-карточек для эффективного запоминания, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для изучения любых тем с использованием интервального повторения.

✨ Уникальные возможности (общие для всех версий)
🃏 Управление карточками: создание, редактирование, удаление карточек (вопрос + ответ).

📚 Категории и теги: группировка карточек по темам для удобной организации.

🧠 Интервальное повторение (Spaced Repetition) — карточки показываются с возрастающим интервалом в зависимости от уровня запоминания (система Лейтнера).

📊 Статистика: количество изученных карточек, процент правильных ответов, уровень прогресса по каждой карточке.

🎯 Режимы обучения:

Классический режим (показ вопроса, затем ответа).

Режим "Угадай по описанию".

Режим экзамена (все карточки подряд без подсказок).

🔄 Импорт/экспорт карточек в CSV и JSON.

💾 Автосохранение прогресса в файл.

🖥️ Интерфейсы:

Командная строка (CLI) с цветным выводом.

Графический интерфейс (GUI) в Python, Java, C#.

Веб-интерфейс в PHP и JavaScript.

Использование (общий синтаксис)
CLI (все версии):

bash
flash add "Столица Франции?" "Париж" --category "География"
flash list --category "География"
flash study --count 10
flash stats
flash export --output cards.csv
Примеры:

Добавить карточку:

bash
python flash_forge.py add "Вопрос" "Ответ" --category "Наука"
Начать изучение 5 карточек:

bash
go run flash_forge.go study --count 5
